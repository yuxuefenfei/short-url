# 短网址管理系统 - 前端

基于Vue 3 + Vite + Pinia + Ant Design Vue的现代化短网址管理系统前端。

## 📖 用户使用手册

### 🎯 系统概述

短网址管理系统是一个简单易用的网址缩短服务，帮助用户将长网址转换为短网址，便于分享和传播。系统提供完整的网址管理、访问统计和用户管理功能。

### 🚀 快速开始

#### 访问系统

**普通用户访问：**
- 主站点：`https://short.ly` 或您的域名
- 管理后台：`https://admin.short.ly` 或您的管理域名

#### 生成短网址

**步骤1：访问主页**

打开浏览器，访问短网址系统主页。

**步骤2：输入长网址**

在主页的输入框中输入您要转换的长网址，例如：
```
https://www.example.com/very/long/url/path/to/content?param1=value1&param2=value2
```

**步骤3：设置选项（可选）**

- **网址标题**：为短网址添加描述性标题
- **过期时间**：设置短网址的有效期（留空表示永不过期）

**步骤4：生成短网址**

点击【生成短网址】按钮，系统会自动生成一个短网址。

**步骤5：使用短网址**

生成成功后，您会看到类似这样的结果：
- **短网址**：`https://short.ly/abc123`
- **短Key**：`abc123`
- **原始网址**：`https://www.example.com/very/long/url...`
- **创建时间**：`2024-01-01 10:00:00`

您可以：
- 点击【复制】按钮复制短网址
- 点击【分享】按钮分享到社交媒体
- 点击【查看详情】查看统计信息

### 📊 查看统计信息

#### 访问统计页面

**方法1：通过生成结果**
生成短网址后，点击【查看详情】直接进入统计页面。

**方法2：手动访问**
访问 `https://short.ly/stats/短网址Key`，例如：
```
https://short.ly/stats/abc123
```

#### 统计信息说明

**基础统计：**
- **总点击次数**：短网址被访问的总次数
- **今日点击**：今天的访问次数
- **创建时间**：短网址的生成时间
- **最后访问**：最近一次访问时间

**详细分析：**
- **访问趋势图**：显示访问量的时间分布
- **来源分析**：显示访问来源（搜索引擎、直接访问等）
- **地理位置**：显示访问者的地区分布（如果支持）

### 🔧 常见问题解答

#### 基础使用问题

**Q：短网址生成失败怎么办？**
A：请检查：
1. URL格式是否正确（必须包含http://或https://）
2. URL是否可达（能正常访问）
3. URL长度是否超过限制（通常2048字符以内）

**Q：短网址可以自定义吗？**
A：目前系统使用自动生成的短Key，暂不支持自定义。如有特殊需求，请联系管理员。

**Q：短网址会失效吗？**
A：短网址默认永不过期，但管理员可以设置过期时间或禁用短网址。

**Q：可以删除已创建的短网址吗？**
A：可以。登录管理后台，在短网址管理页面可以删除自己创建的短网址。管理员可以删除所有短网址。

#### 账户相关问题

**Q：如何注册账户？**
A：请联系系统管理员获取注册权限，目前暂不开放自助注册。

**Q：忘记密码怎么办？**
A：请联系系统管理员重置密码。

**Q：账户被禁用了怎么办？**
A：请联系系统管理员启用账户。

#### 统计和数据分析

**Q：统计数据多久更新一次？**
A：统计数据实时更新，访问发生后立即计入统计。

**Q：可以导出统计数据吗？**
A：可以。在管理后台的短网址管理页面，支持导出CSV格式的统计数据。

**Q：为什么统计数字不准确？**
A：可能原因：
1. 浏览器缓存导致重复访问未计入
2. 网络问题导致访问记录丢失
3. 数据统计延迟（通常不会发生）

#### 技术问题

**Q：短网址访问速度慢怎么办？**
A：短网址系统设计为高性能服务，通常响应时间<100ms。如果遇到访问慢的情况：
1. 检查网络连接
2. 清除浏览器缓存
3. 联系管理员检查系统状态

**Q：短网址被用于垃圾信息怎么办？**
A：系统有安全防护机制：
1. 自动检测恶意URL
2. 用户举报机制
3. 管理员可以禁用违规短网址

**Q：API接口如何使用？**
A：系统提供完整的RESTful API，详细文档请参考API文档。

### 🔒 安全使用指南

#### 安全建议

**用户安全：**
1. **密码安全**：使用强密码，定期更换
2. **账户保护**：不要在公共场所保存登录状态
3. **URL安全**：不要点击来源不明的短网址

**管理员安全：**
1. **权限管理**：合理分配用户权限
2. **操作审计**：定期检查操作日志
3. **系统监控**：关注系统异常情况

#### 风险提示

**短网址的风险：**
1. **隐藏真实URL**：短网址隐藏了真实目标，可能被用于钓鱼
2. **过期风险**：短网址可能随时失效
3. **重定向风险**：短网址可能被修改指向

**防范措施：**
1. **来源验证**：只点击可信来源的短网址
2. **预览功能**：部分服务提供URL预览
3. **安全软件**：使用安全软件检测恶意链接
